## Featured Projects

### [RFA Test Data Analyser](https://github.com/prajwalbekal/rfa-test-analyser)

Automated multi-channel sensor analysis pipeline for rotating component test engineering.

- Processes vibration, force, torque, and temperature data
- Applies Butterworth filtering and FFT analysis
- Detects abnormal frequency peaks, including bearing-defect signatures
- Generates a structured PDF test report with plots and pass/fail verdicts

### [GNC Engineering Student Projects](https://github.com/prajwalbekal/gnc-engineering-student-projects)

Portfolio-ready Guidance, Navigation, and Control projects built from scratch for engineering students.

- 3-axis spacecraft attitude-control simulator with quaternion dynamics
- Extended Kalman Filter for noisy IMU/GPS sensor fusion
- Arduino closed-loop PID controller with desktop plant simulation
- Reproducible validation metrics for tracking error, RMSE, and stability
